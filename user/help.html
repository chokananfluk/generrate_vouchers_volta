ดาวน์โหลดไฟล์
1
สร้างคำขอ Voucher

ไปที่เมนู Voucher ของฉัน แล้วกดปุ่ม + สร้าง Voucher จากนั้นกรอกข้อมูลดังนี้

มูลค่า Voucher
เลือกจากค่าที่กำหนด (฿20, ฿50, ฿100, ...) หรือระบุเองในช่อง "ระบุเอง"
จำนวน Voucher
ระบุจำนวนใบที่ต้องการ (สูงสุด 10,000 ใบต่อครั้ง)
วันเริ่ม / หมดอายุ
กำหนดช่วงเวลาที่ Voucher ใช้ได้ (ไม่จำเป็นต้องระบุ)
ระยะหมดอายุหลังใช้
จำนวนวันที่ Voucher หมดอายุนับจากวันที่ใช้งานครั้งแรก
2
แนบใบเสร็จชำระเงิน

หลังสร้างคำขอแล้ว สถานะจะเป็น รอแนบใบเสร็จ กดปุ่ม 📎 แนบใบเสร็จ เพื่ออัพโหลดไฟล์หลักฐานการชำระเงิน

⚠️ หมายเหตุ: รองรับไฟล์ PDF, JPG, PNG ขนาดไม่เกิน 10MB หากถูกตีกลับสามารถแนบใบเสร็จใหม่ได้
3
รอการอนุมัติจาก Admin

สถานะจะเปลี่ยนเป็น รออนุมัติ ระบบจะส่งอีเมลแจ้งเตือนไปยัง Admin โดยอัตโนมัติ

✅ อนุมัติ — ระบบสร้างรหัส Voucher ให้ทันที
✗ ตีกลับ — แก้ไขและแนบใบเสร็จใหม่ได้

คุณสามารถ ยกเลิก คำขอได้ตราบใดที่ Admin ยังไม่ได้ดำเนินการ

4
Export ไฟล์ CSV

เมื่อได้รับการ อนุมัติแล้ว กดปุ่ม ⬇ Export CSV เพื่อดาวน์โหลดไฟล์รหัส Voucher

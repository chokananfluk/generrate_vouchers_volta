1
จัดการ Voucher

ไปที่เมนู จัดการ Voucher เพื่อดูรายการคำขอทั้งหมด กรองด้วยสถานะหรือค้นหาด้วยชื่อผู้ขอ

✅ การอนุมัติ
กดปุ่ม "อนุมัติ" — ระบบสร้างรหัส Voucher ทันที และส่งอีเมลแจ้งผู้ใช้
✗ การตีกลับ
กดปุ่ม "ตีกลับ" — ต้องระบุเหตุผล ระบบส่งอีเมลแจ้งให้แก้ไขและส่งใหม่
⚠️ หมายเหตุ: ตรวจสอบใบเสร็จก่อนกดอนุมัติเสมอ หากคลิก "ดูใบเสร็จแนบ" ควรตรวจสอบให้แน่ใจว่าถูกต้อง

3
การตั้งค่าระบบ (Setup)

ก่อนใช้งานครั้งแรก ต้องตั้งค่า Google Apps Script Properties และสร้าง Admin user เริ่มต้น

// Script Properties ที่ต้องตั้ง
SUPABASE_URL = "https://xxx.supabase.co"
SUPABASE_SERVICE_KEY = "eyJ..."
ADMIN_EMAIL = "admin@yourcompany.com"
APP_URL = "https://your-app-url.com"
📋 ขั้นตอน:
  1. สร้าง Supabase project และ run schema.sql
  2. สร้าง Storage bucket ชื่อ receipts (Public)
  3. Upload ไฟล์ gas/*.gs ไปยัง Google Apps Script
  4. ตั้ง Script Properties ตามด้านบน
  5. Deploy as Web App (Execute as: Me, Access: Anyone)
  6. รัน function setupAdminUser() เพื่อสร้าง Admin account
  7. แก้ไข js/config.js ให้ใส่ GAS_URL ที่ได้มา
